# 游戏行业日报 Game Digest

每天自动聚合游戏行业内容，AI 评分筛选，直接推送到你的通讯工具（或在对话中显示）。

**理念：** 信息太多，时间有限。日报帮你把值得看的内容先捞出来，你只需要从一份干净的列表开始看。

## 你会得到什么

每日更新的内容，覆盖：

- **微信公众号**：75 个游戏行业公众号（分析、出海、买量、产品拆解）
- **海外媒体**：Deconstructor of Fun、Naavik、GameLook 等
- **播客**：小宇宙游戏类播客
- **搜索**：Google News + 重点游戏关键词

每篇文章附中文标题、两句话摘要、来源和分类，已过滤广告和低质量内容。

## 快速开始

1. 在你的 Claude Code 中安装此 skill：
   ```bash
   git clone https://github.com/xiongxiaoqiu77-prog/game-digest.git ~/.claude/skills/game-digest
   ```
2. 说"看今天的游戏日报"或执行 `/game-digest`
3. Agent 自动拉取最新内容，按你的偏好整理摘要推送给你

不需要任何 API key，也不需要部署任何服务。

## 修改偏好

通过对话即可调整日报风格，直接告诉你的 agent：

- "只看深度拆解类内容"
- "重点关注出海和买量方向"
- "摘要写得更简短一些"
- "推送到飞书"

## 内容分类

| 分类 | 内容类型 |
|------|---------|
| 深度拆解 | 玩法机制、关卡设计、商业化、LiveOps、数值、出海策略 |
| 行业资讯 | 市场动态、新游发布、数据报告 |
| 泛游戏内容 | 游戏文化、玩家社区、周边话题 |

## 工作原理

1. 中心化 feed 每日更新，聚合所有信息源的最新内容（微信公众号通过本地 RSS 服务，海外媒体通过 RSS，播客通过小宇宙 API）
2. 你的 agent 获取 feed——一次 HTTP 请求，不需要 API key
3. 你的 agent 根据你的偏好将内容重新整理为摘要
4. 摘要推送到你的通讯工具（或直接在对话中显示）

## 更新频率

每天北京时间 08:10 左右更新。
